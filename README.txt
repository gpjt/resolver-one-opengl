Resolver One / OpenGL Demos
===========================

These files are the spreadsheets and the Python source for the
OpenGL animations described in these blog posts:

http://www.gilesthomas.com/?p=96
http://www.gilesthomas.com/?p=97

They run in Resolver One, for which you can download a free eval
version here:

http://www.resolversystems.com/products/resolver-one/

In order to run them, you need to download and install the Tao
framework, an open source .NET binding for OpenGL.  You can get
it here:

http://sourceforge.net/projects/taoframework/
